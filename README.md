# 在线收款系统

一个简单的移动端在线收款系统，支持微信和支付宝支付。

## 功能特点

- 移动端友好的界面设计
- 支持微信支付和支付宝
- 可自定义收款金额和说明
- 生成付款二维码
- 简洁易用的用户界面

## 技术栈

- Vue 3
- Vite
- QRCode.vue

## 如何使用

1. 克隆本仓库
2. 安装依赖：`npm install`
3. 开发模式运行：`npm run dev`
4. 构建生产版本：`npm run build`

## 部署

项目配置了自动部署到 GitHub Pages 的工作流。只需将代码推送到 main 分支，GitHub Actions 将自动构建并部署到 gh-pages 分支。

## 注意事项

本项目仅生成模拟的支付链接，实际应用中需要对接微信和支付宝的真实支付 API。
